# RKA Asrama Putra

Berikut adalah Rencana Kerja Anggaran (RKA) untuk unit Asrama Putra yang diekstrak dari dokumen referensi:

| BIDANG | PROGRAM | KEGIATAN | PELAKSANA | SASARAN | PRIORITAS | INDIKATOR KEBERHASILAN |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **SEKRETARIS** | Optimalisasi Manajemen Pengarsipan | Mengelola surat statis & dinamis, mutasi santri, arsip proposal/laporan, backup dokumen Naqieb | Sekretaris Asrama | Pihak terkait, Santri, TU | Program Tetap | Dokumen terarsip & terdokumentasi dengan rapi |
| | Optimalisasi ATK & Sarpras | Pengadaan ATK, sarpras kantor, pemeliharaan sarpras, inventarisasi aset, seragam pengurus | Sekretaris Asrama | Sekretariat, Pengurus | Program Tetap | Kebutuhan operasional kantor terpenuhi |
| | Manajemen Buku Admin | Pengadaan, pengisian rutin, dan evaluasi kelengkapan buku administrasi | Sekretaris Asrama | Sekretariat | Program Tetap | Administrasi asrama terkontrol & lengkap |
| | Database Santri | Update data semesteran & digitalisasi database santri | Sekretaris Asrama | Santri | Program Tetap | Database akurat & mudah diakses |
| | Layanan & Komunikasi | WAG Ortu, booklet profil, buku santri, penyambutan santri baru, optimasi IG asrama | Sekretaris Asrama | Santri, Ortu, Publik | Program Tetap | Informasi & layanan tersampaikan dengan baik |
| | Koordinasi Rapat | Rapat pekanan, rapat terbatas, rapat Naqieb, Rapat Kerja (Raker) | Sekretaris Asrama | Pengurus, Naqieb | Program Tetap | Koordinasi antar pengurus berjalan lancar |
| **BENDAHARA** | Sistem Keuangan | Penyusunan RAB, pencairan dana, pencatatan BKU, pelaporan realisasi | Bendahara Asrama | Pengurus, Yayasan | Program Tetap | Laporan keuangan akuntabel & transparan |
| | Manajemen Aset | Penitipan uang santri, pengadaan sarpras kebutuhan santri | Bendahara Asrama | Santri | Program Tetap | Layanan keuangan santri terjamin |
| **PENDIDIKAN** | Kegiatan Pendidikan | KISS (Kajian Senin Subuh), Halaqah Masa, Bimbel sore, Rapot Asrama Bulanan | Bidang Pendidikan | Santri | Program Tetap | Kegiatan edukasi berjalan konsisten |
| **KESANTRIAN** | Penegakan Disiplin | Operasi rambut/kerapihan, sidak kamar, pembinaan santri, reward & punishment | Bidang Kesantrian | Santri | Program Tetap | Karakter & disiplin santri terbentuk |
| | Minat Bakat | Muhadharah (Pidato), Olahraga pekanan, Seni Bela Diri | Bidang Kesantrian | Santri | Program Tetap | Potensi santri terfasilitasi |
| **PONDOK TAHFIDZ** | Program Tahfidz | Setoran hafalan harian, Tasmi', Munaqasyah, Wisuda Tahfidz | Unit Tahfidz | Santri | Program Tetap | Target hafalan santri tercapai |
| **IBADAH** | Pembiasaan Ibadah | Shalat berjamaah 5 waktu, Tahajjud bersama, Puasa Sunnah | Bidang Ibadah | Santri | Program Tetap | Terbentuknya habit ibadah yang kuat |
| **KEBERSIHAN** | Lingkungan & Kesehatan | Roan (Kerja bakti), pengelolaan sampah, layanan Poskestren, sosialisasi PHBS | Bidang Kebersihan | Santri | Program Tetap | Lingkungan asrama bersih & sehat |
| **SARPRAS** | Pemeliharaan | Perbaikan sarana rusak, pembersihan fasilitas (Masjid, Kamar Mandi, Halaman) | Bidang Sarpras | Fasilitas Asrama | Program Tetap | Sarana prasarana terawat & layak guna |
